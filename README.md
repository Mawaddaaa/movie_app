# 🎬 Movie App

A Flutter movie application built using TMDB API that displays popular movies, movie details, supports Firebase push notifications, and native deep linking without using external deep link plugins.

---

## Features

- Splash Screen
- Popular Movies List
- Movie Details Screen
- Firebase Push Notifications (FCM)
- Native Deep Linking without plugins
- API headers and responses logging in debug mode
- Deep link logging in debug mode
- Clean architecture and expressive method names

## Tech Stack
- Flutter
- Bloc / Cubit
- Dio
- Firebase Messaging
- GoRouter
- TMDB API
